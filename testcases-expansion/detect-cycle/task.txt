Given head, the head of a linked list, determine if the linked list has a cycle in it.

A cycle exists if some node in the list can be reached again by continuously following the next pointer. Internally, the pos is used to denote the index of the node that the tail's next pointer is connected to (0-indexed). pos is not passed as a parameter.

Return true if there is a cycle, otherwise return false.

The linked list node has this structure:
- ListNode class with val (int) and next (ListNode or None)

Example:
Input: head = [3,2,0,-4] with cycle at position 1 (node with value 2)
Output: true (tail connects to the node at index 1)

Input: head = [1] with no cycle (pos = -1)
Output: false

Input: head = [1,2] with cycle at position 0 (node with value 1)
Output: true

Constraints:
- The number of the nodes in the list is in the range [0, 10^4]
- -10^5 <= Node.val <= 10^5
- pos is -1 or a valid index in the linked list
