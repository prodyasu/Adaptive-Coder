Given the root of a binary tree, invert the tree and return the root.
Mirror left/right at every node.

Example:
Input:     4      Output:     4
          / \           / \
         2   7   →     7   2
        / \ / \       / \ / \
       1  3 6  9     9  6 3  1

Constraints:
- The number of nodes in the tree is in the range [0, 100]
- -100 <= Node.val <= 100
